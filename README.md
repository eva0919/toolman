## Installation

### RubyGem
Add this line to your application's Gemfile:

    gem 'toolman'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install toolman

### Clone from github
Step1: Clone this gem to your dir

    git clone

	gem build debugman.gemspec

Step1: Gem install

	gem install ./debugman-0.0.1.gem
